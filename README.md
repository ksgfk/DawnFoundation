## Introduction/介绍
* Minecraft Forge 前置模组，放些有用的小工具
* 使用：在build.gradle添加以下语句
```
allprojects {
    repositories {
		maven { url 'https://jitpack.io' }
	}
}

dependencies {
    implementation 'com.github.ksgfk:DawnFoundation:version'
}
```
* 详情请看[wiki](https://github.com/ksgfk/DawnFoundation/wiki)
