# DawnFoundation/黎明基础
[![](https://jitpack.io/v/ksgfk/DawnFoundation.svg)](https://jitpack.io/#ksgfk/DawnFoundation)
## Introduction/介绍
* 使用：在build.gradle添加以下语句
```
allprojects {
    repositories {
		maven { url 'https://jitpack.io' }
	}
}

dependencies {
    implementation 'com.github.ksgfk:DawnFoundation:version'
}
```
